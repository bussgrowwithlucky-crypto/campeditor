Reference span: `{subjects=[player], setting=[stadium], category=sports,
mood=[epic, mysterious], energy=medium, lighting=low-key, shot_type=wide,
camera_motion=dolly, depth_of_field=deep}`.

  0.625  vibe=0.00  cinema=0.00  | folder='CR7 Winning'     shot=medium  light=low-key

**Difference**: the legacy scorer returns three CR7 Winning medium-low-key

### Scenario 2 — NBA close-up handheld high-key reference
